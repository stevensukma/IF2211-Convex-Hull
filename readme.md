# IF2211-Convex Hull
## Description
Given n random dots, find outer dots that form convex hull. 
The code implement divide and conquer algorithm (quick sort and quick hull)
## How to use
```sh
python3 convex-hull.py
```
## Requirements
```sh
pip3 install requirements.txt
```
